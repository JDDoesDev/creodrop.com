# Welcome to Creodrop

Creodrop is a managed Drupal hosting platform. Launch a fully functional Drupal site in under two minutes with no server configuration, no command line, and no DevOps experience required.

## What Creodrop Does

- **Instant Drupal sites** -- Create a working Drupal installation with a single click
- **Managed infrastructure** -- We handle servers, SSL certificates, databases, and backups
- **Multiple tiers** -- From simple personal sites to full developer environments with SSH and git access
- **Custom domains** -- Connect your own domain with automatic SSL provisioning
- **Control panel** -- Manage all your sites from a clean, modern dashboard

## Who Creodrop Is For

**Non-technical users** who want a professional Drupal site without the complexity of hosting and server management. Pick a tier, create a site, and start building.

**Developers** who want managed Drupal hosting with full CLI access, git-based deployments, and separate development, staging, and production environments.
